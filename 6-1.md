# **6. Теги**
Для создание тегов используется комманда `git tag`. <br>
Следующие опции позволяют:
  + `-l` или `--list` - просомтреть список тегов;
  + `-m` - вносить комментарий к тегу;
  + `-a` - делает тег аннотированым;
  + `-d` -удаляет тег.
